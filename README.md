Tugas ini merupakan pengembangan aplikasi desktop Post Manager dengan mengintegrasikan frontend berbasis Python menggunakan PySide6 dan backend berbasis Laravel REST API.

Aplikasi dirancang agar mampu melakukan pengelolaan data secara efisien melalui antarmuka grafis yang interaktif. Proses komunikasi data dilakukan menggunakan protokol HTTP sehingga pengguna dapat mengambil, menambahkan, memperbarui, maupun menghapus data langsung dari server.

> Integrasi Frontend dan Backend, yaitu koneksi antara aplikasi PySide6 dengan REST API Laravel untuk proses pengambilan dan pengiriman data.
> Manajemen Data (CRUD) yang ditampilkan dalam tabel terstruktur dengan kolom ID, Title, Author, dan Status.
> Optimasi Responsivitas Aplikasi, menggunakan teknik multi-threading melalui QThread agar aplikasi tetap responsif saat melakukan proses fetching data.
